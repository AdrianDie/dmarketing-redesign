Komme i gang første gang

Fra ingenting til en fungerende arbeidsflyt. Set av 20–30 minutter.

I denne modulen setter du opp alt du trenger for å jobbe med nettsiden din. Du gjør det én gang — etterpå er det bare å bruke.

Steg 1 — Opprett GitHub-konto

GitHub er stedet der alle filene til nettsiden din lagres. Tenk på det som en Dropbox-mappe for kode — men med én viktig bonus: GitHub husker alle endringer som noensinne er gjort, for alltid.

1

Gå til github.com

Åpne nettleseren og skriv github.com i adressefeltet. Klikk «Sign up» øverst til høyre.

2

Fyll inn e-post, passord og brukernavn

Velg et brukernavn du husker — det vil inngå i adressen til nettsiden din mens du setter opp alt. Bruk gjerne firmanavnet ditt.

3

Bekreft e-postadressen

GitHub sender en e-post med en kode. Skriv inn koden på skjermen. Velg «Free»-planen — det er den gratis varianten, og det er alt du trenger.

4

Aksepter invitasjon fra Dietrichs

Du har fått en e-post fra GitHub med tittelen «You've been invited to collaborate». Åpne den og klikk «Accept invitation». Nå har du tilgang til repoet ditt — mappen med alle filene til nettsiden din.

Ikke funnet e-posten? Sjekk spam-mappen, eller kontakt oss så sender vi en ny invitasjon.

5

Bekreft at du ser filene

Gå til github.com og logg inn. Du skal nå se et «repository» (en mappe) med navn som ligner på firmanavnet ditt. Klikk på det og sjekk at du ser filer som index.html, style.css og lignende. Da er GitHub-tilgangen i orden.

Steg 2 — Installer VS Code

VS Code er editoren du redigerer filer i. Den er gratis, brukes av millioner av folk verden over, og har alt du trenger innebygd: filviser, koderedigering og git (så du kan sende endringer til GitHub direkte).

1

Last ned VS Code

Gå til code.visualstudio.com. Siden registrerer automatisk om du er på Windows eller Mac og tilbyr riktig versjon. Klikk den store nedlastingsknappen.

2

Installer

Windows: Åpne .exe-filen du lastet ned og klikk deg gjennom installasjonen. Standardvalgene er fine — du trenger ikke endre noe.

Mac: Åpne .zip-filen, dra «Visual Studio Code» til Applications-mappen din.

3

Åpne VS Code

Start programmet. Du ser en velkomstskjerm med noen snarveier. Du trenger ikke gjøre noe her ennå.

Steg 3 — Last ned nettsiden til PC-en

Nå skal du hente filene fra GitHub og lagre dem lokalt på PC-en din. Dette kalles å «klone» — du lager en kopi av mappen fra GitHub til din egen maskin.

1

Finn repo-URL-en på GitHub

Gå til github.com og åpne repoet ditt. Klikk den grønne knappen «<> Code». Kopier URL-en som vises under «HTTPS» — den ser ut som https://github.com/brukernavn/firma-nettside.git.

2

Klon via VS Code

I VS Code: trykk Ctrl+Shift+P (Mac: Cmd+Shift+P) for å åpne kommandopaletten. Skriv Git: Clone og trykk Enter. Lim inn URL-en og trykk Enter igjen.

VS Code spør deg om å velge en mappe på PC-en din — velg et sted du husker, f.eks. Dokumenter. Klikk «Open» når VS Code spør om du vil åpne den klonede mappen.

3

Du ser filene i VS Code

I panelet til venstre (Explorer) skal du nå se alle filene til nettsiden: index.html, style.css, bilder osv. Disse filene ligger nå på PC-en din.

Steg 4 — Se nettsiden lokalt

Nå skal du åpne nettsiden i nettleseren din slik at du kan se endringene du gjør med en gang — før du sender dem live.

1

Åpne index.html i nettleseren

I VS Code: høyreklikk på index.html i Explorer-panelet til venstre → velg «Reveal in File Explorer» (Windows) eller «Reveal in Finder» (Mac). Dobbelklikk på index.html i vinduet som åpner seg. Nettsiden åpner seg i nettleseren din.

Når du har gjort en endring: lagre filen i VS Code (Ctrl+S), og trykk F5 (eller Ctrl+R) i nettleseren for å se den oppdaterte siden.

💡 Vil du at nettleseren oppdaterer seg automatisk når du lagrer?

En utvidelse kalt Live Server gjør at nettleseren oppdaterer seg selv hver gang du trykker Lagre — uten at du trenger å trykke F5. Praktisk, men helt valgfritt.

Slik installerer du den:

  1. I VS Code: klikk ikonet med fire firkanter i sidepanelet til venstre (Extensions)
  2. Søk etter «Live Server»
  3. Klikk «Install» på den av Ritwick Dey
  4. Høyreklikk på index.html i Explorer-panelet → «Open with Live Server»
  5. Nettleseren åpner og oppdaterer seg automatisk fra nå av

Steg 5 — Velg din arbeidsmetode

Her skiller veiene seg. Begge metodene fungerer — forskjellen er pris og hvor mye du gjør manuelt.

Første gang? Les kort hva de to alternativene betyr, og velg det som passer deg. Du kan bytte når som helst — valget huskes til neste gang.

Velg din arbeidsmetode

🟡 Gratis · Mer manuelt

Last ned Claude-appen

I stedet for å åpne en nettleser-fane bruker du Claude-appen — en vanlig app du installerer på PC-en din. Den er gratis og fungerer akkurat som nettsiden, men er lettere å jobbe med siden du har den åpen ved siden av VS Code.

1

Last ned Claude-appen

Gå til claude.ai/download i nettleseren. Klikk nedlastingsknappen for Windows eller Mac. Installer appen som du ville installert et hvilket som helst program.

2

Logg inn eller opprett konto

Åpne Claude-appen. Klikk «Sign Up» og registrer deg med Google-kontoen din eller e-post og passord. Velg «Free»-planen — den er gratis og mer enn god nok til å starte.

3

Du er inne

Du ser en chat-boks. Prøv å skrive «Hei, hvem er du?» — Claude svarer på norsk. Gratis-versjonen har en dagsgrense for antall meldinger, men til vanlig nettstedbruk er det mer enn nok.

Slik fungerer arbeidsflyten din

Siden gratis Claude ikke kan koble seg direkte til GitHub, er du broen mellom de to. Det er noen steg mer enn Claude Code — men helt overkommelig når du har gjort det noen ganger:

  1. VS Code — åpne filen du vil endre, marker alt (Ctrl+A) og kopier (Ctrl+C)
  2. Claude-appen — ny samtale, lim inn koden og beskriv hva du vil endre
  3. Claude svarer med ny, komplett fil — kopier svaret
  4. VS Code — marker alt, lim inn, trykk Ctrl+S
  5. Nettleseren — trykk F5, sjekk at det ser riktig ut
  6. VS Code Source Control — commit + push, og endringen er live om 30–60 sek

Første test — prøv det nå

1

Åpne index.html i VS Code

Klikk på index.html i Explorer-panelet. Filen åpner seg i editoren til høyre.

2

Kopier hele filen

Klikk inne i kodefeltet. Trykk Ctrl+A (Mac: Cmd+A) for å markere alt, deretter Ctrl+C for å kopiere.

3

Åpne Claude-appen — ny samtale

Klikk «New conversation». Lim inn koden (Ctrl+V) og skriv meldingen din, f.eks.:

Her er koden til nettsiden min: [lim inn koden]. Kan du endre telefonnummeret i bunnen til 99 88 77 66?

4

Kopier Claudes svar

Claude svarer med den komplette, oppdaterte filen. Klikk «Copy»-knappen over kodeblokken (eller marker alt og kopier manuelt).

5

Lim inn i VS Code og lagre

Tilbake i VS Code: klikk inne i filen, trykk Ctrl+A og deretter Ctrl+V for å erstatte innholdet med Claudes versjon. Trykk Ctrl+S for å lagre.

6

Sjekk i nettleseren

Gå til nettleserfanen med den lokale siden og trykk F5. Du skal se den nye telefonnummeret på siden. Ser det riktig ut?

7

Send til GitHub — gjør det live

I VS Code: klikk ikonet med tre prikker forbundet med linjer i sidepanelet til venstre (Source Control). Du ser index.html under «Changes».

  1. Klikk «+» ved siden av filen for å «stage» endringen
  2. Skriv en kort melding i feltet øverst, f.eks. «Oppdaterte telefonnummer»
  3. Klikk «Commit», deretter «Sync Changes» (eller «Push»)

Vent 30–60 sekunder — gå til nettstedet ditt live og bekreft at endringen er der. Det fungerte! 🎉

🔵 Enklere · Claude Code

Kjøp Claude Pro

1

Gå til claude.ai

Skriv claude.ai i nettleseren. Opprett en konto (eller logg inn) og klikk «Upgrade to Pro». Prisen er $20 per måned (~220 kr). Du kan si opp når som helst.

2

Fyll inn betalingsinformasjon

Skriv inn kortinformasjonen din og bekreft abonnementet. Du betales månedlig og kan si opp på «Settings → Billing».

Installer Node.js

Claude Code kjøres i terminalen, og for å installere det trenger du Node.js — et lite program som lar deg kjøre verktøy fra kommandolinjen. Det er enkelt å installere.

3

Last ned Node.js

Gå til nodejs.org. Klikk den store knappen merket «LTS» (den anbefalte, stabile versjonen). Last ned og installer — klikk deg gjennom med standardvalgene.

4

Sjekk at Node er installert

Åpne terminalen i VS Code: trykk Ctrl+Ø (Windows, norsk tastatur) eller Ctrl+` (Mac). Skriv node --version og trykk Enter. Du skal se et versjonsnummer som v20.x.x. Hvis du ser det, er Node installert.

Installer Claude Code

5

Skriv installasjonskommandoen

I terminalen (fremdeles i VS Code): skriv denne kommandoen og trykk Enter:

npm install -g @anthropic-ai/claude-code

Det laster ned og installerer Claude Code. Kan ta 1–2 minutter.

6

Logg inn

Skriv claude i terminalen og trykk Enter. Første gang starter en påloggingsprosess — følg instruksjonene på skjermen. Du logger inn med samme konto som du bruker på claude.ai.

Første test — prøv det nå

7

Sørg for at du er i prosjektmappen

Terminalen i VS Code er allerede i mappen du åpnet (prosjektmappen med nettsidefilene). Du kan se stien øverst i terminalen. Skriv ls (Mac) eller dir (Windows) og trykk Enter — du skal se index.html i lista.

8

Start Claude Code

Skriv claude og trykk Enter. Claude Code starter og leser filen CLAUDE.md som Dietrichs har lagt inn i prosjektet ditt — den forklarer Claude hva nettsiden er, hvem bedriften er og hva den skal og ikke skal endre.

9

Be Claude om en liten endring

Skriv noe som dette i chatten og trykk Enter:

Kan du endre telefonnummeret i bunnen av forsiden til 99 88 77 66?

Claude leser filene selv, finner riktig sted og viser deg nøyaktig hva den vil endre.

10

Godkjenn endringen

Claude viser en diff — en oversikt over hva som endres (rødt = fjernes, grønt = legges til). Ser det riktig ut? Skriv y og trykk Enter for å godkjenne. Claude skriver filen.

11

Sjekk i nettleseren

Trykk F5 i nettleserfanen med den lokale siden. Du skal se det nye telefonnummeret. Ser det riktig ut?

12

Send til GitHub — gjør det live

Tilbake i Claude Code-chatten: skriv

Ser bra ut. Commit og push disse endringene.

Claude committer og pusher til GitHub. Vent 30–60 sekunder — gå til nettstedet ditt live og bekreft at endringen er der. 🎉

Du er i gang. Det du nettopp gjennomførte er nøyaktig det du gjør hver eneste gang fremover — det er bare kortere og kortere for hvert gang.

Hva du lærte i denne modulen
  • GitHub-konto opprettet og koblet til repoet ditt
  • VS Code installert og nettsiden klonet til PC-en din
  • Du kan se endringer lokalt før de går live
  • Du har gjort en ekte endring fra start til slutt
