Komme i gang første gang

GitHub-konto og VS Code på plass — grunnlaget for alt videre. Sett av 15–20 minutter.

I denne modulen oppretter du en GitHub-konto og installerer VS Code. Det gjør du én gang — etterpå er verktøyene klare. I del 2 (modul 04b) henter du ned filene og gjør din første endring.

Steg 1 — Få tilgang til nettsiden din på GitHub

GitHub er stedet der alle filene til nettsiden din lagres. Du har allerede fått en invitasjon fra oss — så du trenger ikke gå til github.com selv. Vi bruker Brattøy Elektriske som eksempel her, men flyten er lik uansett hva firmaet ditt heter.

1

Finn invitasjonsmailèn

Sjekk e-posten du fikk da du startet dette kurset. Se etter en mail fra noreply@github.com med emnelinjen «[navn] invited you to collaborate». Finner du den ikke? Sjekk søppelpost-mappen.

Klikk på «View invitation» i mailen.

2

Opprett GitHub-konto

Du blir sendt til GitHub og bedt om å opprette en konto. Fyll inn samme e-postadresse som du fikk invitasjonen på, velg et passord og et brukernavn du husker. Bruk gjerne firmanavnet ditt som brukernavn — f.eks. brattoy.

3

Bekreft e-postadressen din

GitHub ber deg om å skrive inn en kode for å bekrefte at e-posten er din. Gå tilbake til innboksen og se etter en mail med tittelen «Your GitHub launch code». Der står en 8-sifret kode — skriv den inn på GitHub-siden.

4

Logg inn

Etter at kontoen er opprettet blir du bedt om å logge inn. Bruk e-postadressen og passordet du nettopp valgte.

5

Gå tilbake til invitasjonsmailèn og aksepter

Nå som du er logget inn: gå tilbake til den opprinnelige invitasjonsmailèn og klikk «View invitation» igjen. Du ser nå en side med to knapper — klikk «Accept invitation».

Overfør repoet til din GitHub-konto

Fyll inn GitHub-brukernavnet du nettopp opprettet, så overfører vi repoet automatisk til deg. Da eier du det selv og kan klone det fritt.

Steg 2 — Installer VS Code

VS Code er editoren du redigerer filer i. Den er gratis, brukes av millioner av folk verden over, og har alt du trenger innebygd: filviser, koderedigering og git (så du kan sende endringer til GitHub direkte).

Se videoguide for Mac-installasjon
1

Last ned VS Code

Gå til code.visualstudio.com.

Klikk den store «Download for Windows»-knappen midt på siden.

Klikk «Download»-knappen øverst til høyre på siden.

Du blir sendt til nedlastingssiden. Klikk den store «Mac»-knappen.

2

Nedlastingen starter

Nettleseren begynner å laste ned installasjonsfilen. Du ser fremdriften nede i nettleseren (eller i et nedlastingsvindu øverst til høyre). Vent til den er ferdig — filen er ca. 176 MB.

Når det står «Ferdig», klikk på selve filnavnet eller raden for å kjøre installatøren.

3

Start installasjonen

Windows spør om du vil kjøre filen — klikk OK eller Kjør. Installasjonsprogrammet åpner seg og viser lisensavtalen. Velg «I accept the agreement» og klikk Next.

Klikk deg gjennom de neste skjermbildene med Next — standardvalgene er fine. På skjermbildet «Select Additional Tasks» trykker du bare Next.

Klikk Install — installasjonen tar et par minutter. Når den er ferdig klikker du Finish. VS Code åpner seg automatisk.

VS Code er installert. Du har nå verktøyet du trenger for å se og redigere filene nettsiden din består av — index.html, style.css, bilder og alt annet. I neste steg henter du selve filene ned til PC-en din.

3

Installer VS Code

Åpne .zip-filen du lastet ned. Dra «Visual Studio Code»-ikonet til Applications-mappen din. Åpne deretter appen fra Applications.

4

Åpne VS Code

Start programmet. Du ser en velkomstskjerm med noen snarveier. Du trenger ikke gjøre noe her ennå — bare la den stå åpen til neste steg.

Hva skjer videre? Adrian overfører nå repoet til GitHub-kontoen din basert på brukernavnet du sendte inn. Du vil få en e-post fra GitHub med tittelen «Accept repository transfer» — klikk lenken og godkjenn. Deretter er du klar for neste del.

Hva du har gjort i denne delen
  • GitHub-konto opprettet og koblet til repoet ditt
  • VS Code installert og klar til bruk
  • GitHub-brukernavn sendt — repo overføres automatisk
