Oppdatere tekst og innhold

Endre overskrifter, brødtekst, åpningstider, telefon, priser. 90 % av oppdateringer er av denne typen.

Noen ord vi bruker i denne modulen. Her er fire begreper fra «nettside-verdenen» — forklart med vanlige ord:

  • Hero = det store toppfeltet øverst på forsiden — det første folk ser, ofte med et stort bilde og en stor overskrift. (Vi sier også «toppbanner».)
  • H1 = den aller største overskriften øverst på siden, altså hovedtittelen. (H2 og H3 er mindre underoverskrifter.)
  • Footer = bunnfeltet helt nederst på siden, der kontaktinfo og «© firmanavn» ofte står. (Vi sier også «bunnen».)
  • CTA = en handlingsknapp som «Ring nå» eller «Bestill tid». (Står for «call to action», men du kan bare tenke «handlingsknapp».)

Du trenger ikke pugge disse — du kan alltid bare skrive «den store overskriften øverst» eller «knappen» til Claude, så skjønner den det. Møter du flere ord du ikke kjenner, slå dem opp i ordlisten.

Tre regler for gode prompts

  1. Vær konkret om HVOR. "Endre teksten på forsiden" er upresist. "Endre H1-overskriften på forsiden, der det står 'Velkommen'" er bra.
  2. Vær konkret om HVA. Skriv den NYE teksten i prompten. Ikke be Claude finne på den selv (med mindre du vil ha forslag).
  3. Si BEKREFTELSE før publisering. Vil du se forslag før det går live? Si: vis meg endringen først, ikke publiser.

Eksempel-prompts som fungerer

Endre tekst

Endre overskriften på forsiden fra "Velkommen til oss" til "Norges raskeste rørlegger i [By]"

Endre kontaktinfo

Bytt telefonnummeret i footer fra 12 34 56 78 til 99 88 77 66. Sjekk om det står andre steder også.

Endre åpningstider

Oppdater åpningstidene på kontakt-siden: Mandag–torsdag 08–16 Fredag 08–14 Lørdag stengt

Endre priser

Bytt prisen på "Service-pakke Standard" fra 1 990 kr til 2 290 kr. Husk å sjekke om prisen står på flere steder (footer, prislister, FAQ).

Slik gjør du det

Velg din arbeidsmetode

🟡 Gratis · Mer manuelt
  1. VS Code — åpne filen du vil endre (f.eks. index.html). Trykk Ctrl+A → Ctrl+C for å kopiere hele innholdet.
  2. claude.ai — start ny chat. Lim inn koden og beskriv endringen: «Her er koden. Kan du endre telefonnummeret i footer til 99 88 77 66?»
  3. Claude svarer med den komplette, oppdaterte filen. Kopier hele svaret.
  4. VS Code — klikk inne i filen, trykk Ctrl+A → Ctrl+V → Ctrl+S for å erstatte og lagre.
  5. Nettleseren — trykk F5. Sjekk at endringen ser riktig ut.
  6. VS Code Source Control — klikk «+» ved filen, skriv commit-melding, klikk «Commit» → «Sync Changes». Live på 30–60 sek. ✓

Husk: lim alltid inn hele filen til Claude — ikke bare den biten du vil endre. Claude trenger kontekst for å finne riktig sted og unngå å ødelegge resten.

🔵 Enklere · Claude Code
  1. VS Code terminal — skriv claude og trykk Enter.
  2. Beskriv endringen på vanlig norsk: «Endre telefonnummeret i footer til 99 88 77 66»
  3. Claude leser filene selv, finner alle steder og viser deg nøyaktig hva den vil endre.
  4. Godkjenn med y. Claude skriver filene og trykk F5 i nettleseren for å sjekke.
  5. Si: «Ser bra ut, commit og push» — Claude sender til GitHub. Live på 30–60 sek. ✓

Pro-tips: Hvis Claude finner teksten på 5 ulike steder, betyr det ofte at du har inkonsistens i innholdet. Be Claude oppsummere ALLE forekomster så du kan rydde i én slynge.

Hvor finner jeg hvilken tekst

Du trenger ikke huske dette — Claude finner det selv. Men hvis du vil ha en grov oversikt:

  • Hero-tekst (teksten i toppfeltet): alltid på forsiden
  • Tjenestebeskrivelser: tjenester-siden, ofte også på forsiden
  • Åpningstider: kontakt-siden + ofte i footer (bunnen)
  • Telefonnummer: footer (bunnen) på alle sider, handlingsknappen øverst (hero-CTA), kontakt-siden
  • Priser: ofte på flere steder — alltid be Claude sjekke alt

Vanlige feil

Du ber om endring, men teksten finnes flere steder

Hvis du sier "endre 'Velkommen' til 'Hei'" og Claude finner ordet 30 ganger på siden, blir det rot. Vær mer spesifikk: "Endre H1-overskriften på forsiden ..." eller gi mer kontekst.

Du glemmer å nevne mobile-versjon

De fleste tekster er like på desktop (vanlig PC- eller Mac-skjerm) og mobil, men noen ganger har de ulik formulering for å spare plass. Hvis du vil endre begge, si: ... gjelder begge versjoner (desktop og mobil).

Du publiserer noe du ikke ville

Si til Claude: angre den siste endringen. Siden går tilbake til slik den var før, på rundt 30 sekunder. Ingen permanent skade — du kan alltid angre.

Hva du lærte i denne modulen
  • En god prompt forteller Claude HVOR på siden og HVA som skal endres — begge deler trengs
  • Vær konkret: «endre telefonnummeret i bunnen» slår «oppdater kontaktinfo» på presisjon
  • Sjekk alltid siden etter en endring med hard reload (Ctrl+Shift+R / Cmd+Shift+R)
